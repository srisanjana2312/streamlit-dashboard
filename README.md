# streamlit-dashboard

My Streamlit dashboard project



\# Housing Data Dashboard



A Streamlit web app to visualize housing data from Chennai. Built as part of FDS Assignment.



\## Features

\- 📊 Interactive data table with `st.dataframe`

\- 📈 Dynamic bar charts for Price vs Area

\- 📁 Upload your own CSV using `st.file\_uploader`

\- 🔥 Built with Streamlit, Pandas, and Plotly
